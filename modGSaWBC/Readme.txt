To use the configuration menu in Next-Gen, make sure to add the line ( modGSaWBCConfig.xml; ) 
in the lists files ( dx11filelist.txt and dx12filelist.txt ) in the folder ( bin\config\r4game\user_config_matrix\pc\ )
or you can simply drag and drop the configs in ( bin\config\r4game\user_config_matrix\pc\ )
and use TW3 Menu Filelist Updater: https://www.nexusmods.com/witcher3/mods/7171
